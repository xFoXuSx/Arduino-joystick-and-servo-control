# Arduino joystick and servo control
This is my code for Arduino servo control with joystick.


#components:
-Arduino uno.
-Joystick with 5 pins.
-Sevro motor (SG90).
-Buzzer.
-Jumpwires.

#how to use:
1- Connect the Arduino board to the laptop and open arduino IDE.
2-Open the code and track the circuit (pay attention to pins).
2-Click on upload and the Arduino is ready to use.
